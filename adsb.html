The Task

Automatic Dependent Surveillance - Broadcast (ADS-B) is a surveillance technology in which an aircraft shares its position and periodically broadcasts it, enabling it to be tracked. It is often used by air traffic control as a secondary form of communication and doesn't require input from the pilots on the other end. However, civilians can also intercept these signals and create a real-time map of plane locations and their data. ADS-B technology is of interest in other projects at AFLCMC, and a greater extension in range to current capabilities would greatly boost other capabilities.

Our Approach

  • Understand and conform to design specifications for payload in regards to weight, electrical load analysis, battery, and broadcasting signals.
  • Utilize a Raspberry Pi and PiAware to run, collect, and store ADS-B data.
  • Design a robust system with scripts in Python to ensure data collection is always being performed, and implement failsafes and visual indicators.
  • Design housing unit and peripherals to hold all sensitive electronics.

The Solution

By utilizing a Raspberry Pi 3 with PiAware software, we were able to intercept and collect ADS-B data at 1090MHz and stream data in comma separated format. Python scripts were designed to constantly check on the data collection process with multi-threading to be able to check active background processes. Data files were generated every minute, periodically updating the time via a GPS to ensure that in the event of a failure, data is retained up to the minute. A crontab was used to automatically run scripts on the bootup of the Pi. All software was designed with the goal of being able to start data collection without the need of a keyboard, and start with the turn of a switch. Additionally, peripheral indicators were triggered within the scripts to display active power, GPS connectivity, and data collection status. A custom housing unit was designed and machined to fit the specifications provided by the testing team. In comforming to their expectations, we minimized the payload weight and mitigated potential hazards through electrical load analysis and using a NiMH battery.
