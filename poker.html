iOS Poker Bankroll Tracker

AUG 2021 - Present

GitHub Repository

Solo Personal Project

  • Jason Dong

Main screen implementation iwth session logging with specific times, dates, and locations of play. Users have the option to filter by location and tournament format as well. Additionally, users can enter the type of game, cash game stackes, tournament format, as well as view winnings and losses for different sessions.



The Task

Poker is one of my favorite things to do outside of studying and working. It is one of my passions and I am always looking to learn more about strategy and learning more about the game as well as watching professional players excel at the game. As a result, I wanted to self learn iOS development, and attempt to implement my own poker bankroll tracker app. My goal is to be able to track and log sessions, report wins/losses and calculate the hourly of a player. The goal was to make an app that would work for both cash games and tournaments. Afterwards, I also wanted to try to implement pre-flop charts and equity calculators and possibly a hand simulator. This app is still currently under development, but a working prototype is finished locally. I hope to publish this app in the Spring of 2022.

My Approach

  • Learn iOS Development. I completed Angela Yu's iOS Development Bootcamp on Udemy and practiced through the course projects.
  • Sketch out implementation goals and features to implement.
  • Phase 1: Implement basic functionality and testing locally. My outlined goals were to first develop session logging to input session data such as location, duration, winnings/losses, and any notes from the session. Additionally, I wanted functionality to filter out sessions by location and other bonus features.
  • Phase 2: Improve UI design to be more colorful and flashy. Afterwards, start implementing features such as pre-flop charts, equity calculators, graphs, and possibly a hand simulator to display equity with simulated hands and cards left in deck.

Progress So Far

So far, I have completed my planned Phase 1 Implementation with features:

  • Working prototype with live version on local iPhone
  • Log and differentiate between cash and tournament sessions
  • Visualize win/losses from each session
  • Filter by location and session types
  • Ability to start a timer at the start of the session, then end a session and log information
  • Take and store notes specific to each session
  • Customizability options with game variants, stakes, and location types
  • Next Up: Phase 2 Implementation
